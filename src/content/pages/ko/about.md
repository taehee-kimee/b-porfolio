Morethan Log에 오신 것을 환영합니다 - Astro로 구축된 현대적인 정적 블로그 템플릿입니다.

이 템플릿은 인기 있는 [morethan-log](https://github.com/morethanmin/morethan-log) Next.js 템플릿에서 영감을 받았지만, Astro를 위해 완전히 재설계되었습니다.

## 기능

- 🚀 Astro로 빌드하여 빠른 성능 제공
- 🌐 완전한 i18n 지원 (한국어와 영어 기본 제공)
- 📱 모든 기기에서 반응형 디자인
- 🎨 깔끔하고 현대적인 인터페이스
- 📝 쉬운 콘텐츠 생성을 위한 Markdown 지원
- 🔍 내장 검색 기능
- 📊 SEO 최적화
- 🌙 다크 테마 지원
- 📲 Notion 연동으로 편리한 글 작성

## 기술 스택

- **프레임워크**: Astro
- **언어**: TypeScript/JavaScript
- **스타일링**: CSS
- **CMS**: Notion
